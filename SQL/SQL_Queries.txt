Q.. write sql query to find sum of salary of employees whose designation is HR.
==> select sum(salary) from employee where designation='HR'

Q.. Write an sql query to find average salary of employee whose designation is PO.
==> select avg(salary) from employee where designation='PO'

Q.. write an sql query to find min salary of employee whose designation is null.
==> select min(salary) from employee where designation is null

Q.. write an sql query to find min salary of employee whose designation is not null.
==> select min(salary) from employee where designation is not null;

Q.. write a sql query to find max salary of employee whose designation is 'HR'
==> select max(salary) from employee where designation='HR'

Q.. write a sql query to find min,max,avg,sum of salary of employee whose designation is 'Dev'
==> select sum(salary) as sum_salary,min(salary) as min_salary,
	max(salary) as max_salary,avg(salary) as avg_salary from employee where designation ='Dev'

Q.. You want to list departments where the average salary is greater than the overall 
	average salary of all employees.
==> select department,avg(salary)as average_salary from employee 
	having avg(salary)>(select avg(salary) from employee)
